Texto marcado

Para marcar um texto assim, use a teg MARK

Letra pequena

Para adinionar letra pequenas assim, use a teg SMAALL

Texto excluído

Texto excluído assim, pode ser lido más não é relevante. Use a teg DEL

Texto inserido

Para inserir uma palavra ou texto assim, use a teg INS

Texto subescrito

Para adcionar numeros ao quadrado assim, X20+3, use a teg SUP (apenas no número)

Texto sobescrito

Para adcionar a forma do H2O basta colora a teg SUB (apenas no numero)

Escrever códigos

Para escrever códigos na tela dipo o do Python, basta usar a teg CODE

Texto pre-formatado

Use a teg PRE, pode ser usada e é recomendada usar com a teg CODE.
